# Colors

Foundational colors to be used by different [themes](/docs/foundations-theme--docs).
These colors are not context-specific, and should not be used directly in components.

All variables are sourced from [Untitled UI](https://www.untitledui.com/components/color-styles).

<div>
  {Object.values(colorPrimitives).map((item) => (
    <div key={item.id} style={{ marginBlock: 40 }}>
      <Heading>{item.name}</Heading>
      <div class="docs-colorSwatch sb-unstyled theme-light">
        {Object.entries(item.values).map(([weight, hex]) => {
          const variable = `${item.id}-${weight}`;
          return (
            <div class="docs-colorItem" key={variable}>
              <div class="color" style={{ backgroundColor: hex }} />
              <div class="info">
                <div class="variable">{variable}</div>
                <div class="hex">{hex}</div>
              </div>
            </div>
          );
        })}
      </div>
    </div>
  ))}
</div>
