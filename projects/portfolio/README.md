# Portfolio

Portfolio site showcasing professional experience and personal projects.

https://joecocco.com

## Contributing

Read the [contributing docs](/docs/CONTRIBUTING.md).

```bash
# start application dev server
npm start portfolio
```

## Build

```bash
# build app with SSG
npm run build:ssg portfolio

# run local build
npx http-server dist/portfolio/browser -p 8080
```

## Deploy

Auto deployed via Netlify ([config](./netlify.toml))
